About

Grow A Garden API allows you to view the stock data for seeds, eggs, cosmetics, and gears. It also allows developers who want to make their own apps to use this API for free. There are no rate limits but the data is cached regardless so there is no point in spamming it. Data is updated roughly every 10 seconds.

Endpoints

/api/data - All data in one place
/api/last_updated - Time when data last updated
/api/seeds - Seed stock data only
/api/eggs - Egg stock data only
/api/gear - Gear stock data only
/api/cosmeitcs - Cosmetic stock data only
